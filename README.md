## Environment Prepare
``~~~~~~~~``
安装模块

```bash
npm install
```

or

```bash
yarn
```
### Start project

```bash
npm start
```
or
```
yarn start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

代码检查
```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
