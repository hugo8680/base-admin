## Environment Prepare
### Install

```bash
npm install
```

or

```bash
yarn
```
### Start project

```bash
npm start
```
or
```
yarn start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
